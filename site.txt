# MOTD Site Configuration

This file defines site-specific customizations for the base framework.
Use with base.txt (structure) and ai.txt (content) to generate the complete site.

## Brand Identity

```
Brand Name: MOTD
Logo HTML: <span>MOTD</span>
Tagline: Message Of The Day
Description: AI assistant via email - no apps, no accounts, just email
```

## Theme Configuration

```
Theme Storage Key: motd-theme
Default Theme: dark
Default Scheme: stone
```

## Color Palette

Stone (H=60 warm neutral) is the default scheme.
See site.css for full color definitions in OKLCH format.

### Dark Mode (Default)
```css
--bg-primary:   oklch(12% 0.01 60)
--bg-secondary: oklch(16% 0.012 60)
--bg-tertiary:  oklch(22% 0.015 60)
--fg-primary:   oklch(95% 0.01 60)
--fg-secondary: oklch(75% 0.015 60)
--accent:       oklch(80% 0.03 60)
--accent-hover: oklch(90% 0.02 60)
```

### Light Mode
```css
--bg-primary:   oklch(98% 0.005 60)
--bg-secondary: oklch(96% 0.008 60)
--bg-tertiary:  oklch(92% 0.012 60)
--fg-primary:   oklch(25% 0.02 60)
--fg-secondary: oklch(45% 0.02 60)
--accent:       oklch(45% 0.05 60)
--accent-hover: oklch(35% 0.06 60)
```
